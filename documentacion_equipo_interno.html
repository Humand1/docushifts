Paso 1: Conexión con la API

  1. Localiza el campo "API Key" en la barra superior izquierda
  2. Pega tu clave de API (se mostrará como asteriscos por seguridad)
  3. Haz clic en "Conectar" y espera la confirmación
  4. Verifica el estado - debe cambiar a "Conectado"
✅ Conexión Exitosa: Verás "Conectado" en la barra de estado y el botón "Cargar Usuarios" se habilitará.
❌ Error de Conexión: Verifica que tu API Key sea correcta y que tengas acceso a internet. Si el problema persiste, contacta al administrador del sistema.

Paso 2: Cargar Empleados

  1. Haz clic en "Cargar Usuarios" (ahora habilitado)
  2. Espera la carga - verás una barra de progreso
  3. Confirma la carga - aparecerá la cantidad de usuarios cargados
  4. Verifica la lista - los empleados aparecerán en el panel izquierdo
💡 Tip: La primera carga puede tomar unos minutos dependiendo del tamaño de tu organización. Los datos se guardan automáticamente para futuras sesiones.

Solución de Problemas

Guía para resolver los problemas más comunes y saber qué hacer cuando algo no funciona como esperado.

Problemas de Conexión

❌ Error: "No se pudo conectar"

Soluciones:

  1. Verificar que la API Key sea correcta
  2. Confirmar acceso a internet
  3. Revisar estado de la API de Humand
  4. Intentar reconectar después de unos minutos
  5. Contactar al administrador si persiste

Problemas de Configuración

⚠️ Error: "Formato de hora inválido"

Soluciones:

  1. Usar formato HH:MM (ejemplo: 09:00)
  2. Verificar que la hora de fin sea posterior a la de inicio
  3. Evitar solapamientos entre horarios del mismo día
  4. Usar plantillas predefinidas como base

Problemas de Envío

⚠️ Error: "Algunos turnos fueron omitidos"

Qué hacer:

  1. Revisar el mensaje detallado del resultado
  2. Verificar que los empleados seleccionados estén activos
  3. Confirmar que las fechas sean válidas
  4. Revisar los logs de backup para más detalles
  5. Intentar reenviar solo los turnos fallidos
